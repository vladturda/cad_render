CAD Render
----------

Drupal 11 module that provides a block to render a 3D CAD model, and presents options to customize the render appearance. This project integrates the Three.js library for 3D render functionality.

##Customization options:
- Camera: type (perspective/orthographic), zoom
- Animation: none, rotate, rotate on hover
- Dimensions: width, height
- Background: color
- Material: default, solid colour
- Transparency: opacity
- Wireframes
